# world-population-distribution-visualization
A Python data visualization project that analyzes the distribution of country populations using the World Bank Total Population dataset and visualizes it with a histogram.

## Overview
This project visualizes the distribution of country populations using the World Bank Total Population dataset (SP.POP.TOTL). A histogram is created with Python to analyze how populations are distributed across countries.

## Dataset
- Source: World Bank Open Data
- Indicator: SP.POP.TOTL (Total Population)

## Technologies Used
- Python
- Pandas
- Matplotlib

## Project Objective
The objective is to explore the global distribution of country populations using data visualization techniques. The histogram highlights that most countries have relatively small populations, while only a few countries have very large populations.

## Output
- Histogram of country populations
- Basic statistical insights
